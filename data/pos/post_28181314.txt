Obrax wundert sich über den Unterschied von Media Interface (MI), Bestellcode 518, und Universal Media Interface (UMI) Original geschrieben von Obrax Hallo an alle, habe mir heute das Handschuhfach angeschaut, ausser einem UMI Eingang, ist da nichts anderes. Der Wagen hat aber die Option Media Interface Code 518. Ich weiss dass ich die Kabel (Consumer Kabel Kit) nachkaufen muss, ich dachte da waere aber noch so eine kleine quadratische Kiste, wie in den Prospekten abgebildet. ... Gruss, Obrax launchcontrol antwortet darauf: Ich denke, Du bringst hier einfach zwei grundverschiedene Schnittstellen durcheinander. Direkt bestellen kannst Du das Media Interface. Dieses erlaubt Dir den Anschluss von diversen Speichermedien, zwecks Audioausgabe, wenn Du dir beim freundlichen das entsprechende Kabel kaufst (im aktuellen Konfigurator ist es jetzt wohl dabei!). Das UMI kannt Du nicht direkt konfigurieren, sondern nur nachträglich nachrüsten lassen ! Hierbei handelt es sich um eine Box, die NAVI/USB/Bluetooth-Streaming etc. an ein Audio 20 bringt. Kostenpunkt ca.€1099,- (nur die Hardware). antares_x schreibt zusätzlich: Das MediaInterface dient dazu, verschiedene externe Geräte (div. Mediaplayer, Ipod, USB-Sticks usw.) an das Autoradio anzuschliessen. Du kannst dann über das APS50 auf die Daten eines Mediaplayers zugreifen und diese steuern. Die Verbindung erfolgt über einen Kabelsatz (USB, Ipod, ...) im Handschuhfach. Das UMI (Universal Media Interface) gibt es nur für das Audio 20 und rüstet es mit den gleichen Funktionen wie das Media Interface aus und zusätzlich noch mit einer einfachen Navigationsfunktion.
