Den Startscreen kann man aendern, ab SW0310 (SW0350) moeglich. (Flash oder Update noetig und moeglich!) --> Somit auch 32GB SD Karte abspielbar. STG56 --> Byte12 Bit0 von ? auf 0 = Serien Screen Bit0 von 0 auf 1 = S5 Screen Bit1 von 0 auf 1 = RS Screen Bit2 von 0 auf 1 = S4 Screen Bit0 & Bit1 von 0 auf 1 = S-line Screen Bit3 bis Bit7 ohne Funktion Die Werte in [] koennt ihr direkt bei Byte12 eingeben, dann muesst ihr nicht ueber lCode2 gehen.
