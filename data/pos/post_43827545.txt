Ab Modelljahr 10 wird das BCM ( Body Control Module ) verbaut, bei diesem kann die LED Kaltlichtdiagnose codiert werden. Folgende Punkte sind Notwendig: Ein Softwarestand des BCM ( Stg 09 ) von min." 0667". Auf Byte 17 Bit 7 muss gesetzt werden ( auf 1 ). Wenn dieser Softwarestand nicht vorhanden ist kann beim Händler ein Firmwareupdate durchgeführt werden, dazu können folgende TPIs verwendet werden. Beim einem Originalsoftwarestand unter 0600 muss erst auf einem Softwarestand von "0615" geupdatet werden. TPI 2021271 (SVM Code 09A001) Nun muss ein weiteres Update erfolgen auf "0667" Dieses ist ab einem Softwarestand von "0615" aufzuspielen. TPI 2029738 (SVM Code 09A008) Die Originalen LED Kennzeichenbeleuchtungen haben folgende Teilenummern: 4H0 943 021 LED Kennzeichenbeleuchtung links 4H0 943 022 LED Kennzeichenbeleuchtung rechts N 905 624 01 Kennzeichenschraube 3,5x10 ( 1x je Seite )
